# Customer Intelligence System using Machine Learning

## Project Overview

This project develops a Customer Intelligence System using supervised and unsupervised machine learning techniques. It predicts customer development categories and performs customer segmentation.

## Algorithms Used

### Classification Models
- Logistic Regression
- Naive Bayes
- K-Nearest Neighbors
- Support Vector Machine
- Decision Tree
- Random Forest

### Ensemble Learning
- AdaBoost
- Gradient Boosting
- XGBoost
- LightGBM
- Stacking Classifier

### Clustering
- K-Means
- DBSCAN
