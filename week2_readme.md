# Week 2 - End-to-End Machine Learning Pipeline on Sales/Price Data

## 📌 Overview

This project demonstrates the implementation of an end-to-end Machine Learning pipeline using Tesla Electric Vehicle Sales and Production data (2015–2025).

The objective is to predict the average selling price of Tesla vehicles while covering the complete machine learning workflow, including preprocessing, exploratory data analysis, feature engineering, regression modeling, hyperparameter tuning, and time series forecasting.

---

## 📂 Dataset

**Dataset:** Tesla EV Deliveries and Production Data (2015–2025)

The dataset contains:

- Year
- Month
- Region
- Model
- Estimated Deliveries
- Production Units
- Average Price (USD)
- Battery Capacity
- Driving Range
- CO₂ Saved
- Charging Stations
- Source Type

---

## 🎯 Project Objectives

- Perform data preprocessing and cleaning
- Conduct exploratory data analysis (EDA)
- Engineer meaningful features
- Encode categorical variables
- Apply feature scaling
- Train regression models
- Perform cross validation
- Tune hyperparameters using GridSearchCV
- Compare model performance
- Analyze feature importance
- Forecast future average prices using time series analysis
