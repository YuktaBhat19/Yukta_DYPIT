# Text Generation using RNN, LSTM and GRU

## Project Overview
This project implements text generation using three Recurrent Neural Network architectures:
- Vanilla RNN
- Long Short-Term Memory (LSTM)
- Gated Recurrent Unit (GRU)

The models are trained on a text corpus to predict the next word in a sequence and generate meaningful text.

---

## Objective
- Learn text preprocessing and tokenization.
- Build and train RNN, LSTM, and GRU models.
- Generate text from a trained model.
- Compare the performance of the three architectures.

---

## Technologies Used
- Python
- TensorFlow / Keras
- NumPy
- Matplotlib

---

## Project Workflow
1. Import required libraries
2. Load text corpus
3. Preprocess text
4. Tokenize text
5. Generate input sequences
6. Build Vanilla RNN model
7. Build LSTM model
8. Build GRU model
9. Train all models
10. Compare training loss
11. Generate sample text
